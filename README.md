# go-cooperhewitt-shoebox

Tools for archiving your Cooper Hewitt shoebox.

## Caveats

This is wet paint. It's probably too soon for you.

## Setup

```
make build
```

## Tools

### shoebox

```
$> ./bin/shoebox -token <COOPERHEWITT_API_TOKEN> -shoebox <PATH_TO_LOCAL_SHOEBOX>
```
